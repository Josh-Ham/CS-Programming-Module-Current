idk package¶

Submodules¶

idk.covid_data_handler module¶

Covid data handling module.

Fuctions: parse_csv_data – gets CSV data from file, process_covid_csv_data – gets specific data from the covid data, covid_API_request – makes a API request to Cov19API, schedule_covid_updates – schedules updates, update_covid_data – gets updated covid data, process_covid_local_dict_data – gets specific local covid data, process_covid_country_dict_data – gets specific national covid data, get_updates – gets uncompleted updates, schedule_check_data – runs scheduler, get_covid_data – gets formatted covid data, remove_data_update – removes future update, set_repeating_data_update – makes an update repeat

idk.covid_data_handler.covid_API_request(location: str = 'nope', location_type: str = 'no') → dict¶

Get up-to-date Covid data as a dictionary.

Optional arguements: location – location to get data from, location_type – type of location to get data from

idk.covid_data_handler.get_covid_data() → dict¶

Get the covid data.

idk.covid_data_handler.get_updates() → list¶

Get the list of updates.

idk.covid_data_handler.parse_csv_data(csv_filename: str) → list¶

Get a list of strings for each line of the file.

Keyword arguements: csv_filename – csv data filename

Return values: csv_data – list of line of the csv file

idk.covid_data_handler.process_covid_country_dict_data(covid_dict_data: dict) → tuple[str, int, int, int]¶

Get the location, weekly infections, hospital cases and deaths.

Keyword arguements: covid_dict_data – national covid data

Return values: location – data location, national_7day_infections – last 7 day infections, hospital_cases – current hospital cases, deaths_total – current death toll

idk.covid_data_handler.process_covid_csv_data(covid_csv_data: list) → tuple[int, int, int]¶

Get cases, hospital cases and deaths from the Covid data.

Keyword arguements: covid_csv_data – list of strings of lines from a csv file

Return values: last7days_cases – number of covid cases in the last 7 days, current_hospital_cases – number of current hospital cases, total_deaths – total number of covid related deaths

idk.covid_data_handler.process_covid_local_dict_data(covid_dict_data: dict) → tuple[str, int]¶

Gets the location and 7 day infections.

Keyword arguements: covid_dict_data – local covid data

Return values: location – data location, local_7day_infections – last 7 days infections
